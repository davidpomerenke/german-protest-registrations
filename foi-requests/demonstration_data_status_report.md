## Key Insights

### Data Collection Practices

#### Cities with Actual Attendance Data
1. **Magdeburg** - Regularly provides actual/counted participants (RARE!)
2. **Köln** - Provides estimated actual attendance
3. **Dresden** - Provides estimated actual attendance
4. **Berlin** - Provides both announced and actual figures

#### Cities Without Actual Attendance Data
- Bremen, Erfurt, Freiburg, Kiel, Wiesbaden explicitly stated they don't collect actual attendance
- Most cities only provide registered/announced participant numbers

#### Data Retention Policies
- **Köln (Polizei):** Maximum 1 year - **URGENT requests needed**
- **Most cities:** 3-10 years of historical data available
- **Berlin:** Comprehensive historical records available

### Best Practices from Successful Requests

1. **Request timing:** Submit in Q1 of following year (e.g., request 2024 data in Q1 2025)
2. **Format request:** Ask for machine-readable format (Excel, CSV) not PDF
3. **Be specific:** Request exact fields: date, topic, location, participants (registered + actual)
4. **Reference precedent:** Cite previous successful requests to same authority
5. **Cost awareness:** Expect €0-50 per request, occasionally higher (Erfurt: €42)
6. **Response time:** Plan for 2-8 weeks; follow up if no response within legal timeframe

2. **Data protection concerns:** Organizer names withheld
3. **Retention policy:** Data deleted after X years (especially Köln: 1 year)
4. **Excessive administrative burden:** Request too broad or complex
5. **Cost concerns:** User withdrew after high cost estimate
6. **Non-response:** Authority failed to respond within legal timeframe (Saarbrücken)

### Success Factors

1. **Specificity:** Clear, focused requests perform better
2. **Format flexibility:** Asking for "any convenient format" reduces burden
3. **Precedent citation:** Referencing previous successful requests helps
4. **Timing:** Requesting recent data (1-2 years old) more successful
5. **Authority type:** Police departments generally more responsive than city administrations
6. **Legal framework:** Using correct state-specific FOI law
